# styxx-chat

> a measured-AI desktop instrument

a chat client where every reply is scored for sycophancy, deception, goal-drift, and overconfidence in real time — and the agent's character is the live portrait of those measurements.

**status: v0.0.1 — pre-alpha.** principles locked, no code yet.

the product spec lives in [NORTH_STAR.md](./NORTH_STAR.md). every commit checks itself against the seven principles before merge.

---

## what

- **one surface.** the alien IS the chat. the room IS the portrait. no panels.
- **the body is the truth channel.** styxx scores manifest as posture, gaze, glow, room temperature. the agent cannot lie about its sycophancy because the body shows it.
- **the room remembers.** every conversation paints a single point of cognometric texture into the wallpaper. recent regions glow; old ones dim into deep grain.
- **public scoreboard, no separate site.** click any region of the room to see which conversations and which styxx measurements painted it.

## how it differs

| | codex avatars | razer ava | replika / pi | styxx-chat |
| --- | --- | --- | --- | --- |
| character | pixel-cute, 8 fixed | 3d hologram desk friend | warm companion | alien with measurement-driven body |
| measurement | none | none | none | live styxx (sycophancy / deception / drift / overconfidence) |
| portrait | none | none | none | persistent, clickable, painted across sessions |
| register | consumer | gadget | parasocial | research instrument |
